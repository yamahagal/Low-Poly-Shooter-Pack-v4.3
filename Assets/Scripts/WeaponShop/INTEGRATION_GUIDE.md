# Интеграция с инвентарем игрока

Эта инструкция объясняет, как настроить автоматическое добавление ВСЕХ оружий из JSON в инвентарь игрока.

## Обзор

Система магазина оружия поддерживает интеграцию с существующей системой инвентаря из Low Poly Shooter Pack через компонент `WeaponShopInventoryIntegration`.

**ВАЖНО**: Начиная с текущей версии, ВСЕ оружия из JSON файла автоматически добавляются в инвентарь игрока при запуске игры, а не только купленные!

## Как работает интеграция

### Автоматический процесс

1. **Загрузка JSON** - сервис `WeaponShopServiceSimple` загружает все оружия из JSON файла
2. **Инициализация интеграции** - компонент `WeaponShopInventoryIntegration` инициализируется
3. **Добавление всех оружий** - ВСЕ оружия из JSON добавляются в инвентарь игрока
4. **Поиск префабов** - для каждого оружия ищется соответствующий префаб
5. **Создание экземпляров** - префабы создаются через `Instantiate`
6. **Добавление в инвентарь** - оружия добавляются к инвентарю игрока
7. **Обновление инвентаря** - инвентарь игрока обновляется и активирует первое оружие

### Компоненты

| Компонент | Назначение |
|-----------|-----------|
| `WeaponShopServiceSimple` | Загрузка оружия из JSON |
| `WeaponShopInventoryIntegration` | Интеграция с инвентарем |
| `Inventory` (из Low Poly Shooter Pack) | Инвентарь игрока |

### Настройка в Unity

#### Шаг 1: Создайте GameManager

1. В Hierarchy: `Right Click > Create Empty`
2. Назовите "GameManager"

#### Шаг 2: Добавьте компоненты

На GameObject "GameManager" добавьте следующие компоненты:

1. **CurrencyService**
   - Путь к сохранению валюты: `saves/currency_save.json`
   - Starting Balance: 1000

2. **WeaponShopServiceSimple**
   - Inventory File Path: `saves/inventory_test.json`

3. **WeaponShopInventoryIntegration**
   - Add All Weapons On Start: ✓ (включено)
   - Initialization Delay: 0.5 (рекомендуется для корректной инициализации)
   - Weapon Prefabs: перетащите префабы оружия:
     - P_LPSP_WEP_Handgun_01.prefab
     - P_LPSP_WEP_SMG_01.prefab
     - P_LPSP_WEP_RL_01.prefab
     - P_LPSP_WEP_Shotgun_01.prefab
     - P_LPSP_WEP_Sniper_01.prefab

**Важно**: Название префаба должно совпадать с `weaponName` в JSON файле!

#### Шаг 3: Проверьте инвентарь игрока

Убедитесь, что на сцене есть GameObject с компонентом `Inventory` (из Low Poly Shooter Pack).

### Как проверить работу

1. Запустите игру
2. Проверьте консоль Unity на сообщения:
   - "Интеграция с Inventory инициализирована. Сервис: WeaponShopServiceSimple"
   - "Добавлено оружий в инвентарь: X, пропущено (уже есть): Y"
   - "Инвентарь обновлен после добавления оружий"

3. Проверьте, что ВСЕ оружия доступны в инвентаре игрока
4. Переключайтесь между оружиями с помощью клавиш 1-5 или колесика мыши

### Требования

Для работы интеграции необходимо:

1. ✅ Компонент `WeaponShopServiceSimple` на сцене
2. ✅ Компонент `CurrencyService` на сцене
3. ✅ Компонент `WeaponShopInventoryIntegration` на сцене
4. ✅ Инвентарь игрока `Inventory` на сцене
5. ✅ Префабы оружия в папке `Assets/Infima Games/Low Poly Shooter Pack/prefabs/Weapons/`
6. ✅ JSON файл с данными оружия в `Assets/saves/inventory_test.json`

### Частые проблемы

#### Компонент не отображается

Если компонент `WeaponShopInventoryIntegration` не отображается в списке:

1. Перезагрузите Unity (`Assets > Reimport All`)
2. Проверьте консоль на ошибки компиляции
3. Убедитесь, что файл `InventoryIntegration.cs` находится в папке `Assets/Scripts/WeaponShop/Integration/`

#### Оружие не добавляется

Если оружие не добавляется в инвентарь:

1. Проверьте, что включено поле `Add All Weapons On Start`
2. Проверьте, что префабы добавлены в поле `Weapon Prefabs`
3. Проверьте, что названия префабов совпадают с `weaponName` в JSON файле
4. Проверьте, что инвентарь игрока `Inventory` существует на сцене
5. Проверьте, что JSON файл содержит данные в `weaponDefinitions`

#### Оружия не переключаются

Если оружия добавлены, но не переключаются:

1. Убедитесь, что инвентарь игрока инициализирован после добавления оружий
2. Попробуйте увеличить `Initialization Delay` в компоненте `WeaponShopInventoryIntegration`
3. Проверьте консоль на сообщение "Инвентарь обновлен после добавления оружий"

### Изменения в поведении

**Ранее**: Оружие добавлялось в инвентарь только после покупки

**Теперь**: ВСЕ оружия из JSON добавляются в инвентарь при запуске игры

Это означает, что:
- ✅ Все оружия из JSON доступны для переключения
- ✅ Статус "purchased" в JSON влияет только на магазин, не на инвентарь
- ✅ Вы можете редактировать JSON файл для добавления новых оружий
- ✅ Оружия автоматически появляются в инвентаре при следующем запуске игры

#### Ошибки в консоли

Если видите ошибки в консоли:

- "Сервис магазина оружия не найден в сцене."
- "Инвентарь игрока не найден в сцене."
- "Префаб оружия [название] не найден."

### Структура JSON файла

```json
{
  "weaponDefinitions": {
    "handgun_01": {
      "weaponName": "Пистолет",
      "weaponType": "Handgun"
    }
  }
}
```

Компонент `WeaponShopInventoryIntegration` ищет префаб по значению `weaponName` в JSON файле.

### Дополнительная информация

**Примечание**: Интеграция работает только с компонентом `WeaponShopInventoryIntegration`. Если вы используете другие компоненты магазина, убедитесь, что они также поддерживают событие `OnWeaponPurchased`.

**Поддерживаемые типы оружия**:
- Handgun (Пистолет)
- SMG (ПП)
- AssaultRifle (Автомат)
- Shotgun (Дробовик)
- Sniper (Снайперская винтовка)
