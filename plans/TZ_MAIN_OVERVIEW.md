# Техническое задание: Система магазина оружия, скинов и обвесов (Обзор)

## Обзор проекта

Создать комплексную систему магазина для игры "Low Poly Shooter Pack v4.3", включающую:
- Покупку оружия
- Покупку скинов с разной редкостью
- Покупку и установку обвесов (6 типов)
- 3D-предпросмотр товаров
- Сохранение прогресса
- Интеграцию с существующими системами

## Технические требования

### Язык программирования
- C#

### Unity версии
- Совместимость с Unity 2020.3+

### Архитектура

**Слои:**
1. **Data Layer** - ScriptableObject конфигурации
2. **Services Layer** - Сервисы магазина
3. **UI Layer** - Компоненты интерфейса
4. **Persistence Layer** - Сохранение/загрузка данных
5. **Integration Layer** - Интеграция с существующими системами
6. **Test Layer** - Тестирование

## Файлы для создания

### Data Layer
- `WeaponData.cs` - ScriptableObject для конфигурации оружия
- `SkinData.cs` - ScriptableObject для конфигурации скина
- `SkinMaterialData.cs` - ScriptableObject для материалов скина
- `AttachmentData.cs` - ScriptableObject для конфигурации обвеса
- `AttachmentType.cs` - Перечисление типов обвесов

### Services Layer
- `ICurrencyService.cs` - Интерфейс сервиса валюты
- `CurrencyService.cs` - Реализация сервиса валюты
- `IWeaponShopService.cs` - Интерфейс сервиса магазина оружия
- `WeaponShopService.cs` - Реализация сервиса магазина оружия
- `ISkinShopService.cs` - Интерфейс сервиса магазина скинов
- `SkinShopService.cs` - Реализация сервиса магазина скинов
- `IAttachmentShopService.cs` - Интерфейс сервиса магазина обвесов
- `AttachmentShopService.cs` - Реализация сервиса магазина обвесов

### UI Layer
- `WeaponShopUI.cs` - Основной UI магазина оружия
- `SkinsShopUI.cs` - Основной UI магазина скинов
- `AttachmentsShopUI.cs` - Основной UI магазина обвесов
- `WeaponCard.cs` - Компонент карточки оружия
- `SkinCard.cs` - Компонент карточки скина
- `AttachmentCard.cs` - Компонент карточки обвеса
- `StatsPanel.cs` - Компонент панели характеристик
- `PreviewPanel.cs` - Компонент панели предпросмотра

### Persistence Layer
- `ShopSaveSystem.cs` - Основная система сохранения
- `ShopSaveData.cs` - Класс данных сохранения
- `ShopBackupSystem.cs` - Система резервного копирования

### Integration Layer
- `ShopIntegration.cs` - Основной класс интеграции
- `InventoryIntegration.cs` - Интеграция с Inventory
- `WeaponIntegration.cs` - Интеграция с Weapon
- `ServiceLocatorIntegration.cs` - Интеграция с ServiceLocator

### Test Layer
- `CurrencyServiceTests.cs` - Тесты сервиса валюты
- `WeaponShopServiceTests.cs` - Тесты сервиса магазина оружия
- `SkinShopServiceTests.cs` - Тесты сервиса магазина скинов
- `AttachmentShopServiceTests.cs` - Тесты сервиса магазина обвесов
- `ShopSaveSystemTests.cs` - Тесты системы сохранения
- `InventoryIntegrationTests.cs` - Тесты интеграции с Inventory
- `WeaponIntegrationTests.cs` - Тесты интеграции с Weapon
