# Техническое задание: Система обвесов оружия (Часть 3 - Обвесы)

## Обзор

Создать систему обвесов (attachments) для оружия в игре "Low Poly Shooter Pack v4.3". Обвесы позволяют модифицировать характеристики оружия.

## Требования

### 1. Типы обвесов

**6 основных типов обвесов:**

1. **Scope (Прицел)** - увеличивает точность на дальних дистанциях
   - Модификаторы: Accuracy (+), Range (+)
   - Визуальное изменение: добавление оптического прицела

2. **Muzzle (Дульный насадок)** - уменьшает отдачу и шум
   - Модификаторы: Accuracy (+), Damage (-)
   - Визуальное изменение: глушитель/компенсатор

3. **Laser (Лазерный целеуказатель)** - улучшает точность при стрельбе с бедра
   - Модификаторы: Accuracy (+)
   - Визуальное изменение: лазерная точка

4. **Grip (Цевье/Рукоятка)** - уменьшает отдачу
   - Модификаторы: Accuracy (+), ReloadTime (-)
   - Визуальное изменение: передняя рукоятка

5. **Magazine (Магазин)** - увеличивает ёмкость магазина
   - Модификаторы: MagazineCapacity (+), ReloadTime (+)
   - Визуальное изменение: увеличенный магазин

6. **Bipod (Сошки)** - улучшает точность при стрельбе из положения лёжа
   - Модификаторы: Accuracy (+), FireRate (-)
   - Визуальное изменение: сошки

### 2. Конфигурация обвесов

**Параметры обвеса:**
- Уникальный идентификатор обвеса (attachment_id)
- Название обвеса
- Тип обвеса (Scope, Muzzle, Laser, Grip, Magazine, Bipod)
- Модификаторы характеристик:
  - Damage (множитель или фиксированное значение)
  - FireRate (множитель или фиксированное значение)
  - MagazineCapacity (фиксированное значение)
  - ReloadTime (множитель или фиксированное значение)
  - Accuracy (множитель или фиксированное значение)
  - Range (множитель или фиксированное значение)
- Стоимость обвеса
- Описание обвеса
- Превью изображение (опционально)
- Префаб 3D модели обвеса (для визуализации)

**Связь с оружием:**
- Каждый обвес привязан к конкретному оружию
- Одно оружие может иметь несколько обвесов одного типа
- Только один обвес каждого типа может быть одновременно установлен

### 3. Магазин обвесов

**Функционал:**
- Отображение списка доступных обвесов для выбранного оружия
- Фильтрация по типам обвесов
- Просмотр превью обвеса на оружии
- Покупка обвеса за игровую валюту

- Отображение статуса (куплено/не куплено/установлено/снято)

**UI элементы:**

- Цена обвеса

- Кнопка "Установить" (если куплено и не установлено)
- Кнопка "Снять" (если куплено и установлено)

**Логика:**

- Сохранение купленных обвесов
- Сохранение установленных обвесов для оружия
- Автоматическое применение установленных обвесов в игре
- Пересчёт характеристик оружия с учётом обвесов

### 4. Система модификаторов характеристик

**Типы модификаторов:**

1. **Множитель (Multiplier)** - умножает базовое значение
   - Пример: Damage * 1.1 = +10% урона
   - Используется для: Damage, FireRate, Accuracy, Range

2. **Фиксированное значение (Fixed)** - добавляет/вычитает фиксированное значение
   - Пример: MagazineCapacity + 10 = +10 патронов
   - Используется для: MagazineCapacity, ReloadTime

**Правила применения:**
- Модификаторы суммируются для всех установленных обвесов
- Финальное значение не может быть меньше минимального порога
- Финальное значение не может превышать максимальный порог

## Технические требования

### Язык программирования
- C#

### Unity версии
- Совместимость с Unity 2020.3+

### Архитектура

**Слои:**
1. **Data Layer** - ScriptableObject конфигурации обвесов
2. **Services Layer** - Сервис магазина обвесов
3. **UI Layer** - Компоненты интерфейса
4. **Persistence Layer** - Сохранение/загрузка данных

### Файлы для создания

**Data Layer:**
- `AttachmentData.cs` - ScriptableObject для конфигурации обвеса
- `AttachmentType.cs` - Перечисление типов обвесов

**Services Layer:**
- `IAttachmentShopService.cs` - Интерфейс сервиса магазина обвесов
- `AttachmentShopService.cs` - Реализация сервиса магазина обвесов

**UI Layer:**
- `AttachmentsShopUI.cs` - Основной UI магазина обвесов
- `AttachmentCategoryUI.cs` - UI категории обвесов
- `AttachmentItemUI.cs` - UI элемент обвеса в списке
- `AttachmentPreview3D.cs` - 3D предпросмотр обвеса на оружии
- `WeaponStatsPanel.cs` - Панель характеристик оружия с учётом обвесов

**Persistence Layer:**
- `AttachmentShopSaveSystem.cs` - Система сохранения для магазина обвесов
