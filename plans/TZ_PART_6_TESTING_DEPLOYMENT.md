# Техническое задание: Тестирование и развёртывание (Часть 6 - Тестирование и деплой)

## Обзор

Создать комплексную систему тестирования для магазина оружия, скинов и обвесов, а также подготовить проект к развёртыванию.

## Требования

### 1. Unit тестирование

**Тестирование сервисов:**
- Тестирование сервиса валюты
- Тестирование сервиса магазина оружия
- Тестирование сервиса магазина скинов
- Тестирование сервиса магазина обвесов

**Примеры тест-кейсов:**
```csharp
// CurrencyService Tests
[Test] void AddCurrency_IncreasesBalance()
[Test] void SpendCurrency_DecreasesBalance()
[Test] void SpendCurrency_InsufficientFunds_ReturnsFalse()

// WeaponShopService Tests
[Test] void BuyWeapon_SufficientFunds_ReturnsTrue()
[Test] void BuyWeapon_InsufficientFunds_ReturnsFalse()
[Test] void BuyWeapon_AlreadyPurchased_ReturnsFalse()
[Test] void SelectWeapon_NotPurchased_ReturnsFalse()

// SkinShopService Tests
[Test] void BuySkin_SufficientFunds_ReturnsTrue()
[Test] void BuySkin_WeaponNotPurchased_ReturnsFalse()
[Test] void SelectSkin_SkinNotPurchased_ReturnsFalse()

// AttachmentShopService Tests
[Test] void BuyAttachment_SufficientFunds_ReturnsTrue()
[Test] void InstallAttachment_AttachmentNotPurchased_ReturnsFalse()
[Test] void CalculateStats_WithAttachments_ReturnsCorrectValues()
```

### 4. Тестирование производительности

**Метрики производительности:**
- Время загрузки магазина
- FPS при работе с магазином
- Память, используемая магазином
- Время сохранения/загрузки данных

**Целевые значения:**
- Время загрузки магазина: < 2 секунды
- FPS при работе с магазином: > 60 FPS
- Память, используемая магазином: < 100 MB
- Время сохранения данных: < 100 мс
- Время загрузки данных: < 200 мс
