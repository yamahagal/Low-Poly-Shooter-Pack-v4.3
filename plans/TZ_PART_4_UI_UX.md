# Техническое задание: UI и UX системы магазина (Часть 4 - Интерфейс)

## Обзор

Создать пользовательский интерфейс для системы магазина оружия, скинов и обвесов в игре "Low Poly Shooter Pack v4.3".

## Требования

### 1. Общий дизайн интерфейса

**Стиль:**
- Минималистичный дизайн в стиле Low Poly
- Тёмная тема с яркими акцентами
- Чёткая иерархия элементов
- Адаптивный интерфейс для разных разрешений

**Цветовая палитра:**
- Фон: Тёмно-серый (#1A1A1A)
- Карточки: Средне-серый (#2D2D2D)
- Акценты: Оранжевый (#FF6B00) - основной цвет
- Текст: Белый (#FFFFFF)
- Текст вторичный: Светло-серый (#B0B0B0)
- Редкость Common: Серый (#808080)
- Редкость Rare: Синий (#3B82F6)
- Редкость Epic: Фиолетовый (#8B5CF6)
- Редкость Legendary: Золотой (#F59E0B)

### 2. Основной экран магазина

**Компоненты:**
1. **Шапка (Header)**
   - Название магазина ("Магазин оружия")
   - Кнопка "Назад" (для выхода из магазина)
   - Отображение баланса валюты

2. **Навигация (Navigation)**
   - Табы: "Оружие", "Скины", "Обвесы"
   - Индикатор активного таба

3. **Основной контент (Main Content)**
   - Список оружия/скинов/обвесов
   - Панель предпросмотра
   - Панель параметров

4. **Футер (Footer)**
   - Информация о версии магазина
   - Кнопка "Настройки" (опционально)

### 3. Экран магазина оружия

**Компоненты:**
1. **Список оружия (Weapon List)**
   - Scroll View с вертикальной прокруткой
   - Карточки оружия с миниатюрами
   - Индикатор купленного оружия
   - Индикатор выбранного оружия

2. **Карточка оружия (Weapon Card)**
   - Изображение оружия
   - Название оружия
   - Тип оружия
   - Стоимость
   - Кнопка "Купить" / "Выбрать"
   - Индикатор редкости (если применимо)

3. **Панель предпросмотра (Preview Panel)**
   - 3D модель оружия
   - Возможность вращения модели
   - Отображение выбранного скина
   - Отображение установленных обвесов

4. **Панель параметров (Stats Panel)**
   - Название оружия
   - Тип оружия
   - Список характеристик с визуальными индикаторами
   - Модификаторы от обвесов (если есть)

### 4. Экран магазина скинов

**Компоненты:**
1. **Выбор оружия (Weapon Selector)**
   - Dropdown или список купленного оружия
   - Отображение текущего выбранного оружия

2. **Список скинов (Skins List)**
   - Grid Layout с горизонтальной/вертикальной прокруткой
   - Карточки скинов с миниатюрами
   - Индикатор редкости скина
   - Индикатор купленного скина
   - Индикатор выбранного скина

3. **Карточка скина (Skin Card)**
   - Изображение скина
   - Название скина
   - Редкость скина
   - Стоимость
   - Кнопка "Купить" / "Выбрать"
   - Цветовой индикатор редкости

4. **Панель предпросмотра (Preview Panel)**
   - 3D модель оружия с применённым скином
   - Возможность вращения модели
   - Отображение разницы с базовым скином

### 5. Экран магазина обвесов

**Компоненты:**
1. **Выбор оружия (Weapon Selector)**
   - Dropdown или список купленного оружия
   - Отображение текущего выбранного оружия

2. **Категории обвесов (Attachment Categories)**
   - Tabs: "Прицелы", "Дульные насадки", "Лазеры", "Цевье", "Магазины", "Сошки"
   - Индикатор активной категории
   - Индикатор количества установленных обвесов

3. **Список обвесов (Attachments List)**
   - Grid Layout с горизонтальной/вертикальной прокруткой
   - Карточки обвесов с миниатюрами
   - Индикатор купленного обвеса
   - Индикатор установленного обвеса

4. **Карточка обвеса (Attachment Card)**
   - Изображение обвеса
   - Название обвеса
   - Тип обвеса
   - Стоимость
   - Список модификаторов характеристик
   - Кнопка "Купить" / "Установить" / "Снять"

5. **Панель характеристик (Stats Panel)**
   - Базовые характеристики оружия
   - Модификаторы от установленных обвесов
   - Финальные характеристики
   - Визуальное сравнение (базовые vs финальные)

### 6. Компоненты UI

**Кнопки (Buttons):**
- Primary Button (основное действие): Оранжевый фон, белый текст
- Secondary Button (вторичное действие): Прозрачный фон, оранжевый текст
- Disabled Button (неактивное действие): Серый фон, серый текст

**Индикаторы (Indicators):**
- Currency Balance: Отображение баланса валюты с иконкой
- Purchase Status: Иконка купленного/не купленного товара
- Selection Status: Иконка выбранного элемента
- Rarity Indicator: Цветовая полоска или рамка

**Анимации (Animations):**
- Hover эффект на карточках товаров
- Нажатие кнопок с эффектом нажатия
- Плавные переходы между экранами
- Анимация появления элементов
- Анимация покупки (успех/ошибка)

### 7. Адаптивность

**Разрешения:**
- Desktop: 1920x1080 (основной)
- Tablet: 1024x768
- Mobile: 1280x720 (минимальный)

**Адаптивные элементы:**
- Grid Layout автоматически адаптируется под размер экрана
- Scroll View работает на всех разрешениях
- Размеры карточек пропорционально изменяются
- Размеры шрифтов пропорционально изменяются

### 8. Доступность (Accessibility)

**Требования:**
- Поддержка навигации клавиатурой
- Поддержка геймпада (опционально)
- Чёткий контраст текста
- Понятные иконки с подписями
- Возможность увеличения масштаба интерфейса

### 9. Локализация

**Поддерживаемые языки:**
- Русский (основной)
- Английский

**Локализуемые элементы:**
- Все текстовые элементы UI
- Названия оружия, скинов, обвесов
- Описания товаров
- Сообщения об ошибках

### 10. Производительность

**Требования:**
- Плавная работа при 60 FPS
- Оптимизация 3D предпросмотра (LOD, batching)
- Кэширование изображений и 3D моделей
- Минимизация перерисовки UI

## Технические требования

### Unity UI Framework
- Unity UI (uGUI) или UI Toolkit
- TextMeshPro для текста
- Scroll Rect для списков
- Grid Layout Group для сеток
- Canvas Scaler для адаптивности
