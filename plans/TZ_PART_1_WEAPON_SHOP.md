# Техническое задание: Система магазина оружия (Часть 1 - Магазин оружия)

## Обзор

Создать систему магазина для покупки оружия в игре "Low Poly Shooter Pack v4.3".

## Требования

### 1. Основной магазин оружия

**Функционал:**
- Отображение списка доступного оружия
- Просмотр параметров оружия перед покупкой
- Покупка оружия за игровую валюту
- 3D-предпросмотр оружия

**Параметры оружия для отображения:**
- Название оружия
- Тип оружия (Handgun, SMG, Shotgun, AssaultRifle, Sniper, RocketLauncher, GrenadeLauncher)
- Урон (Damage)
- Скорострельность (Fire Rate - выстрелов в минуту)
- Ёмкость магазина (Magazine Capacity)
- Время перезарядки (Reload Time)
- Точность (Accuracy - чем меньше, тем точнее)
- Дальность стрельбы (Range)
- Стоимость оружия

**UI элементы:**
- Список оружия (Scroll View)
- Карточка оружия с превью
- Кнопка "Купить" (если не куплено)
- Кнопка "Выбрать" (если куплено)
- Отображение баланса валюты
- Панель параметров выбранного оружия

**Логика:**
- Проверка достаточности валюты перед покупкой
- Списание валюты при покупке
- Сохранение купленного оружия в системе сохранений
- Отображение статуса (куплено/не куплено)

### 2. Система валюты

**Функционал:**
- Отображение текущего баланса валюты
- Добавление валюты игроку
- Проверка достаточности средств
- Сохранение баланса в системе сохранений

**Начальный баланс:** 1000 единиц валюты

### 3. Сохранение прогресса

**Формат данных (JSON):**
```json
{
  "currency": 1000,
  "weapons": {
    "weapon_id_1": {
      "purchased": true,
      "selected": true
    },
    "weapon_id_2": {
      "purchased": false,
      "selected": false
    }
  }
}
```

### 4. Интеграция с существующими системами

**Интеграция с Inventory:**
- Добавление купленного оружия в инвентарь игрока
- Возможность выбора оружия для игры

**Интеграция с ServiceLocator:**
- Регистрация сервисов магазина и валюты

## Технические требования

### Язык программирования
- C#

### Unity версии
- Совместимость с Unity 2020.3+

### Архитектура

**Слои:**
1. **Data Layer** - ScriptableObject конфигурации оружия
2. **Services Layer** - Сервисы магазина и валюты
3. **UI Layer** - Компоненты интерфейса
4. **Persistence Layer** - Сохранение/загрузка данных

### Файлы для создания

**Data Layer:**
- `WeaponData.cs` - ScriptableObject для конфигурации оружия

**Services Layer:**
- `ICurrencyService.cs` - Интерфейс сервиса валюты
- `CurrencyService.cs` - Реализация сервиса валюты
- `IWeaponShopService.cs` - Интерфейс сервиса магазина оружия
- `WeaponShopService.cs` - Реализация сервиса магазина оружия

**UI Layer:**
- `WeaponShopUI.cs` - Основной UI магазина оружия
- `WeaponItemUI.cs` - UI элемент оружия в списке
- `WeaponPreview3D.cs` - 3D предпросмотр оружия
- `WeaponParametersPanel.cs` - Панель параметров оружия
- `CurrencyBalanceUI.cs` - Отображение баланса валюты

**Persistence Layer:**
- `WeaponShopSaveSystem.cs` - Система сохранения для магазина оружия
